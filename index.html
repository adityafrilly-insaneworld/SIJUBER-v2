🔒 Firebase Security Rules
Paste rules berikut ke Firebase Console → Realtime Database → Rules. Dengan rules ini, HANYA user yang sudah login via Firebase Auth yang bisa membaca/menulis data.
{ "rules": { "sijuber": { "data": { ".read": "auth != null", ".write": "auth != null" }, "users": { ".read": "auth != null", ".write": "auth != null" } } } }
